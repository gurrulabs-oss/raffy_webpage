Demande de Suppression des Donnees - Raffy

Si vous avez utilise l'application Raffy et souhaitez que vos donnees personnelles soient supprimees definitivement de nos systemes, vous pouvez envoyer une demande par email.

Notes importantes

  • Si vous avez utilise Raffy sans creer de compte, toutes vos donnees sont stockees uniquement sur votre appareil. Nous n'y avons pas acces et ne pouvons pas les supprimer a distance.
  • Si vous vous etes connecte avec votre compte Gmail, certaines donnees peuvent etre stockees de maniere securisee dans notre base cloud (via Supabase), et nous pouvons les supprimer sur demande.

Comment demander la suppression

Envoyez un email a : gurrulabs@gmail.com

Objet : Data Deletion Request - Raffy

Merci d'inclure :

  • L'adresse email utilisee pour la connexion (si applicable).
  • Toute autre information utile pour nous aider a retrouver vos donnees.

Nous traiterons votre demande sous 7 jours et confirmerons une fois les donnees supprimees.

Pour toute question, contactez-nous a gurrulabs@gmail.com.
